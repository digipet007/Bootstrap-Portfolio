Grouper Personality Testing

The Personality Inventory application is designed to provide users with opportunities for introspection by taking a pseronality test and comparing their test results to others in their group. Crystal API is used to complete the surveys and return the student's personality archetype. Users must create a profile within Crystal prior to taking the test. The data is stored in a relational database and each class is given a group name. To view graphical and tabular results for each class, click on Group Results, and select your group name from the drop-down menu.
